###### Markdown列表语法
----------------------

### 无序列表

使用 `*`、`+` 或 `-` 作为列表标记。

| Markdown 语法 | 渲染效果 |
|--------------|----------|
| <code>`* 项目一`</code> | <ul><li>项目一</li></ul> |
| <code>`+ 项目二`</code> | <ul><li>项目二</li></ul> |
| <code>`- 项目三`</code> | <ul><li>项目三</li></ul> |

#### 示例

```markdown
* 水果
  * 苹果
  * 香蕉
  * 橙子

* 蔬菜
  + 胡萝卜
  + 西兰花
```

#### 渲染效果

* 水果
  * 苹果
  * 香蕉
  * 橙子

* 蔬菜
  + 胡萝卜
  + 西兰花

---
### 有序列表

使用数字加句点 `.` 作为列表标记。

#### 示例

```markdown
1. 第一步
2. 第二步
3. 第三步
```

#### 渲染效果

1. 第一步
2. 第二步
3. 第三步

#### 重要提示

**Markdown 会自动重新编号**，所以你可以都写成 `1.`：

```markdown
1. 第一步
1. 第二步
1. 第三步
```

渲染效果同样是：

1. 第一步
1. 第二步
1. 第三步

---
### 嵌套列表

在子列表前添加 **4 个空格** 或 **1 个 Tab** 进行缩进。

#### 示例

```markdown
1. 饮料
    * 咖啡
    * 茶
2. 食物
    * 面包
    * 水果
        * 苹果
        * 香蕉
```

#### 渲染效果

1. 饮料
    * 咖啡
    * 茶
2. 食物
    * 面包
    * 水果
        * 苹果
        * 香蕉

### 列表中包含其他元素

列表项可以包含**段落、代码块、引用**等其他元素。

#### 示例

```markdown
* 这是一段文字。

    这是同一列表项的第二段。

* 这个列表项包含代码：

    ```python
    print("Hello, World!")
    ```

* 这个列表项包含引用：

    > 这是引用内容
```

#### 渲染效果

* 这是一段文字。

    这是同一列表项的第二段。

* 这个列表项包含代码：

    ```python
    print("Hello, World!")
    ```

* 这个列表项包含引用：

    > 这是引用内容

---
### 注意事项

1. **分隔列表**
   - 列表前后需要**空行**与其他内容分隔
   - 不同类型的列表之间也要有空行

2. **缩进一致性**
   - 同一级别的列表项要对齐
   - 嵌套列表建议使用 **4 个空格**缩进（兼容性最好）

3. **列表项内容**
   - 列表标记后加**空格**再写内容
   - 如果列表项是多段内容，后续段落需要**缩进对齐**  

4. **转义列表**
   - 如果不想显示为列表，可以用反斜杠转义：`\1. 文本`

5. **兼容性**
   - 无序列表的三种标记（`*` `+` `-`）效果完全相同
   - 建议在文档中**统一使用一种**，推荐使用 `-`
